------------文件介绍------------
【LiveReload】是桌面应用程序的源代码

【解放F5】是Chrome浏览器扩展插件的源代码

【解放F5.crx】是生成好的Chrome浏览器扩展插件


------------使用方法------------
1.将【解放F5.crx】安装到Chrome中

2.打开【LiveReload】里编译好的【解放F5】桌面应用程序

3.在程序中选择目录，然后开始监听

4.刷新调试页面，开始工作啦。。。


------------注意事项------------
默认端口为23333
